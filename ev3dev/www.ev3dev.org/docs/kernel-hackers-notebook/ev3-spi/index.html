    Serial Perepherial Interface

    Edit on Github

    The AM1808 SoC has 2 hardware Serial Peripheral Interface Bus controllers (SPIs).

    Note: CS stands for Chip Select, which allows more than one device to be connected to a bus.

    Usage

    SPI0:

    • CS0: NAND Flash (Not implemented in ev3dev)
    • CS3: Analog/Digital converter

    SPI1: (chip select not used)

    • LCD controller
