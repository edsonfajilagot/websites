Exercise 11: Moving a File (mv)

In this exercise you learn how to move a file from one location to another using the mv command.

Do This

Linux/OSX

$ cd temp
$ mv awesome.txt uncool.txt
$ ls
newplace    uncool.txt
$ mv newplace oldplace
$ ls
oldplace    uncool.txt
$ mv oldplace newplace
$ ls
newplace   uncool.txt
$
